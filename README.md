# 🐘 泰有料 ThaiYouLiao Chatbot  
一個模擬泰式餐廳服務流程的 Python 終端機聊天機器人，包含訂位、查詢菜單、優惠資訊、外帶等多種功能。

## 📌 功能總覽（8 大功能）

### **1️⃣ 餐廳訂位 / 查詢 / 取消**
- 顯示所有時段剩餘訂位  
- 新增訂位（自動產生訂位編號）  
- 查詢訂位資訊  
- 取消訂位（釋放時段座位）
A
### **2️⃣ 優惠活動查詢**
- 顯示本月優惠（午餐 9 折、新會員禮、外帶滿額贈飲）

### **3️⃣ 菜單查詢（多關鍵字搜尋）**
- 支援關鍵字查詢：辣、不辣、海鮮、前菜、甜點等  
- 支援多關鍵字（例如「辣 海鮮」）  
- 回傳符合條件的菜單品項

### **4️⃣ 門市資訊 / 客服電話**
- 顯示門市地址  
- 客服電話  
- 營業時間  

### **5️⃣ 外帶預訂（保留舊版功能）**
- 外帶資料構已建立  
- 尚未啟動新功能流程  

### **6️⃣ 會員制度說明**
- 消費滿額可累積點數  
- 點數折抵規則說明  

### **7️⃣ 食物保存與配送資訊**
- 建議保存方式  
- 配送滿額免運規則  

### **8️⃣ 離開對話**
- 結束服務  
- 或詢問是否繼續進行下一輪功能操作

## 🛠️ 系統架構

### 📁 主要資料結構
| 名稱 | 用途 |
|------|------|
| `self.reservations` | 訂位資料（以電話為 key） |
| `self.time_slots` | 全時段座位資訊（每 30 分鐘） |
| `self.menu` | 菜單資料（含標籤） |
| `self.takeaway_orders` | 外帶訂單資料 |
| `self.all_keywords` | 可查詢的菜單關鍵字 |
